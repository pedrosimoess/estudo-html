Código fonte / Pré formatação

Exemplo de um código em python

num = int(input('digite um número')) if num % 2 == 0: print(f'número{num} é par') else: print(f'0 número {num} é impar') print('FIM DO PROGRAMA')

Ultilizando a tag pre temos a escrita do código exatamente como ele foi digitado, até mesmo com identação.


    num = int(input('digite um número'))
    if num % 2 == 0:
        print(f'número{num} é par')
    else:
        print(f'0 número {num} é impar')
    print('FIM DO PROGRAMA')
