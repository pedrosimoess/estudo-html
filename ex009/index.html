Trabalhando com Listas

Listas Ordenadas

Esse é o tipo de lista onde a ordem importa

  1. Acordar
  2. Tomar café
  3. Ligar para a mãe
  4. Escovar os dentes

Listas Não-Ordenadas

Esse é o tipo de lista onde a ordem NÃO importa

  • Acordar
  • Ligar para a mãe
  • Tomar Café
  • Escovar os dentes

Listas Mistas

Podemos também ter listas mistas, configurando listas dentro de listas

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4

Lista de Definições

É um tipo de lista em HTML que funciona com uma proposta parecida com a de um dicionário, onde temos os termos e suas descrições

HTML
Linguagem de marcação utilizada para criar conteúdo de sites
CSS
Linguagem de marcação utilizada para a especificação de estilos em sites
JavaScript
Linguagem de programação para criar interatividade em sites
